# FAQ

## О проекте

Бот для анализа игроков CS2 на Faceit. Помогает находить тиммейтов.

## Бесплатно?

Базовые функции бесплатно. Есть платные подписки с расширенными возможностями.

**Нельзя:**
- Коммерческое использование
- Копировать код
- Создавать копии проекта
