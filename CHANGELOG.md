# Changelog

Все значимые изменения в проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [0.2.0] - 2024-12

### Добавлено
- Система обработки ошибок с кастомными исключениями
- Модели базы данных (User, Subscription, Payment, DemoAnalysis)
- Настройка SQLAlchemy ORM
- Unit тесты для всех сервисов (~40+ тестов)
- Integration тесты для API endpoints (~20+ тестов)
- CI/CD pipeline с GitHub Actions
- Централизованная конфигурация API для frontend
- Валидаторы данных для демо-файлов
- Dependency Injection для сервисов
- Конфигурация pytest с маркерами
- Фикстуры и моки для тестирования
- .env.example файл с примерами настроек
- Документация по улучшениям (IMPROVEMENTS_SUMMARY.md)
- Роадмап проекта (ROADMAP.md)

### Изменено
- Интегрированы все роуты в главное приложение
- Улучшена структура проекта
- Обновлена конфигурация settings.py с валидацией
- Убраны хардкод URL из frontend компонентов
- Улучшена обработка ошибок в сервисах
- Обновлены зависимости (SQLAlchemy 2.0, pydantic-settings)

### Исправлено
- Дублирование кода в settings.py
- Проблемы с импортами
- Структура обработки ошибок
- Конфигурация проекта

### Удалено
- Неиспользуемые файлы и дублирование кода

## [0.1.0] - 2024-12

### Добавлено
- Первый релиз проекта
- Базовая функциональность AI-ассистента
- Расширение для браузера
- Docker деплой
- Базовая структура проекта
- Система подписок
- Система платежей
- Поиск тиммейтов
- Анализ демо-файлов
